# Activity 5.2A — Context first — given a chart, write the context read before identifying any pattern
## What you're doing
You are practicing the habit of reading the market before reading the setup.
## What you need
A chart and a space to write the context.
## Your task
Describe the macro backdrop, session structure, and price structure before naming a pattern.
## What to look for
Note whether the market is trending, ranging, showing liquidity, or reacting to news.
