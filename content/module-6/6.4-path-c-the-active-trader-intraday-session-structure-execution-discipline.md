# Path C — The active trader (intraday, session structure, execution discipline)
## The idea in one sentence
This path focuses on short-term sessions, fast decisions, and tight execution.
## Why this matters
If you can commit to the session rhythm, this path gives more frequent opportunities.
## How it works
You trade 1H and below, watching session structure, order flow, and execution risk closely.
> Key concept: Intraday trading is as much about discipline as it is about pattern recognition.
## What this looks like in the market
A trader watching the market open, the session highs and lows, and the moves between them.
## Common mistake
Letting emotion or speed override the process.
