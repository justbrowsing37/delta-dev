# Identifying swing setups using Module 5 confluence framework
## The idea in one sentence
You should only trade swing setups that have multiple supporting factors.
## Why this matters
Confluence reduces the odds of entering a weak, one-dimensional trade.
## How it works
You combine trend, range structure, liquidity, and invalidation into one setup.
> Key concept: Swing setups are stronger when daily and 4H context align.
## What this looks like in the market
A setup that has a tested level, a range breakout, and a clean invalidation point.
## Common mistake
Taking a setup that looks okay on one chart but ignores the bigger picture.
