# mvp/ — Product App (Flask)

This folder is the **MVP planet** — the private Flask application that runs Delta One.

## Intent

All product code (Flask app, routes, models, templates, services) lives in `app/` at the repo root.
The `mvp/` folder is reserved for MVP-specific resources: seed scripts, migration notes,
environment setup guides, and any staging/review tooling.

## Structure

```
mvp/
  README.md       — this file
  seeds/          — data seed scripts
  notes/          — build notes, decisions log
```
